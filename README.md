## Requirements:


[x] Connect to a Ethereum Metamask wallet.

[x] Display your Ethereum balance.

[x] Send a transaction of "x" ETH where x is a user-defined input.


## Instructions

Open the repository

In a new terminal run:

`yarn install` then `yarn dev`
